HEADER - Psylum

A full-featured level, with objectives and puzzles. You are an Agent, sent to retrieve Files from a supposedly abandoned Lab– but when you arrive, it’s as if the building itself is aware of your presence… Built in UDK. All level design, art, models, textures, animation, scripting, lighting done by myself. Features include: Objectives UI which complete after each task is completed; lockpicking puzzle UI built in Flash, and imported via Scaleform. Custom scripting was made in Kismet to allow for the player to pick up an object and move it around. Gameplay video:
